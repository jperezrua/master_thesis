Template for the MSc Thesis for the VIBOT master


The files (among others):

- Thesis.tex: main file which includes the other tex files.
- Header.tex: contains the header (title, abstract, TOC,..).
- Intro.tex: a chapter explaining guidelines.
- Background.tex: a sample chapter to check margins, ...
- Appendix.tex: a sample appendix.
- Refs.bib: an example of bib file for building bibliography with bibtex.
- Thesis.pdf/.dvi: the final version of the thesis in dvi and pdf formats.


Robert Martí, March 2008.
